Overview

A wrapper for the data returned by the PoolFixity class that can be used in a space-efficient rack response. It returns a CSV document with the fixity data provided by a PoolFixity object.

def each
  yield '"name","silo","sha1","md5","time","status"' + "\n"
  @pool_fixity.each do |r|
    yield [r.name, r.silo, r.sha1, r.md5, r.time.to_s, r.status.to_s].map{ |e| StoreUtils.csv_escape(e) }.join(',') + "\n"
  end
end
