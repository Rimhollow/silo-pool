Overview

List all of the files from a silo (except for .lock files). Data is returned as a list of arrays, where the first element (key) is the path; and the second element (value) is a struct containing .path, a string; .mtime, a Time object; and .size, a fixnum.

def initialize filesystem
  @index = 0
  @list  = []

  Find.find(filesystem) do |path|
    next if File.directory? path
    next if path =~ /\.lock$/
    stat = File.stat path
    @list.push [ path, Struct::FindStreamRecord.new(path, stat.size, stat.mtime) ]
  end
  @list.sort! { |a, b|  a[0] <=> b[0] }
end
